1Initialize Git Repository

Open terminal in your project folder and run:

git init

2Add All Files

git add .

3Create First Commit

git commit -m "Initial commit: Productivity Tracker app"

4Create GitHub Repository

  1. Go to github.com and sign in
  2. Click the "+" icon in the top right
  3. Select "New repository"
  4. Name it: productivity-tracker (or any name you like)
  5. DO NOT check "Initialize with README" (we already have one)
  6. Click "Create repository"

5Connect to GitHub

After creating the repository, GitHub will show you commands. Use these (replace YOUR_USERNAME with your GitHub username):

git remote add origin https://github.com/YOUR_USERNAME/productivity-tracker.git
git branch -M main
git push -u origin main

6Verify

Go to your GitHub repository page and you should see all your files! üéâ
