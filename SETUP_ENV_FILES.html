üìç Where to Find Your Credentials

  1. Go to supabase.com and log in
  2. Select your project
  3. Click Settings (gear icon) in the left sidebar
  4. Click API Keys
  5. You'll see:
On the API Keys page, you'll find:
  • Project URL - At the top of the page (looks like https://xxxxx.supabase.co)
  • Publishable key - In the "Publishable key" section (starts with sb_publishable_...)
  • Secret key - In the "Secret keys" section (starts with sb_secret_...) - Click "Reveal" to see it
