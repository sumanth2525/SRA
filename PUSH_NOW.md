# Push to GitHub - Quick Guide

## Option 1: Create New Repository (Recommended)

1. Go to https://github.com/sumanth2525
2. Click the **"+"** icon → **"New repository"**
3. Repository name: `productivity-tracker` (or any name you like)
4. Description: "Full-stack productivity tracking app with React and Python"
5. Choose **Public** or **Private**
6. **DO NOT** check "Initialize with README"
7. Click **"Create repository"**

Then run these commands:

```bash
git remote add origin https://github.com/sumanth2525/productivity-tracker.git
git branch -M main
git push -u origin main
```

## Option 2: Use Existing SRA Repository

If you want to push to your existing SRA repository:

```bash
git remote add origin https://github.com/sumanth2525/SRA.git
git branch -M main
git push -u origin main
```

**⚠️ Warning:** This will overwrite the SRA repository if it has different content!
