Setup Commands & Instructions

1Supabase Setup

First, you need to set up your Supabase database:

üìù Instructions:
  1. Go to supabase.com and create a free account
  2. Create a new project (wait ~2 minutes for it to be ready)
  3. Go to SQL Editor in the left sidebar
  4. Click New Query
  5. Copy and paste the contents of supabase_schema.sql
  6. Click Run to execute
  7. Go to Settings ‚Üí API to get your credentials

2Create Environment Files

Create environment files with your Supabase credentials:

# Create .env file in root directory
Note: Create a file named .env in the root directory with this content:
VITE_SUPABASE_URL=https://your-project-id.supabase.co VITE_SUPABASE_ANON_KEY=your-anon-key-here
# Create backend/.env file
Note: Create a file named .env in the backend directory with this content:
SUPABASE_URL=https://your-project-id.supabase.co SUPABASE_KEY=your-anon-key-here SUPABASE_SERVICE_KEY=your-service-role-key-here

# Kill all node processes (if needed)
# Check if servers are running
Common Issues:
  • Missing Supabase credentials: Make sure both .env files are created with correct values
  • Port already in use: Close other applications using ports 3000 or 8000
  • Module not found: Run npm install again
  • Database errors: Make sure you ran the SQL schema in Supabase
