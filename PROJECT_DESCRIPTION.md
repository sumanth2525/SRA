# Productivity Tracker - Project Description

## Short Description (for GitHub)
```
Full-stack productivity tracking application with React frontend and Python FastAPI backend. Track health, finances, notes, investments, and passwords in one place.
```

## Medium Description (for GitHub About section)
```
A comprehensive personal productivity and tracking application built with React and Python FastAPI. Features health tracking (protein intake, calories), finance management with analytics, notes & reminders, investment portfolio tracking, and secure password management. Uses Supabase as the database with a modern, mobile-responsive UI.
```
